#Ekososyal - Çevreci Sosyal Web Uygulaması
Ekososyal kullanıcıların çevresel farkındalıklarını artırmayı hedefleyen, sosyal etkileşim özellikleri içeren bir web uygulamasıdır.
Bu uygulamanın frontend kısmı HTML, CSS ve JavaScript kullanılarak geliştirilmiştir. 
Bu repository’de, uygulamanın backend ve veritabanı entegrasyonu yapılmadan önce LocalStorage kullanılarak tasarlanmış ve çalışır durumda olan sürümü yer almaktadır.
##Anasayfa
Navigasyon sistemi sayesinde diğer sayfalara erişim sağlanır ve kullanıcı deneyimi merkezi bir yapı üzerinden yönetilir.
