# Ekososyal - Çevreci Sosyal Web Uygulaması
Ekososyal kullanıcıların çevresel farkındalıklarını artırmayı hedefleyen, sosyal etkileşim özellikleri içeren bir web uygulamasıdır.
Bu uygulamanın frontend kısmı HTML, CSS ve JavaScript kullanılarak geliştirilmiştir. 
Bu repository’de, uygulamanın backend ve veritabanı entegrasyonu yapılmadan önce LocalStorage kullanılarak tasarlanmış ve çalışır durumda olan sürümü yer almaktadır.
## Anasayfa
Navigasyon sistemi sayesinde diğer sayfalara erişim sağlanır ve kullanıcı deneyimi merkezi bir yapı üzerinden yönetilir.

## Temel Bilgiler
Bu sayfada orman yangınları, küresel ısınma ve çeşitli çevresel olaylar hakkında bilgilendirici içerikler sunulmaktadır.
Amaç, kullanıcıların çevresel sorunlar hakkında bilinçlenmesini sağlamak ve farkındalık oluşturmaktır.

## Etkinlikler
Etkinlikler sayfasında giriş yapan kullanıcılar: “Duyuru Yap” butonu ile kendi çevre etkinliklerini paylaşabilir, Var olan etkinliklere yorum yapabilir, Topluluk etkileşimine katkıda bulunabilir. Bu sayfa dinamik kullanıcı etkileşimini destekleyen bir sosyal paylaşım alanı olarak tasarlanmıştır.
